On the ABS algorithm with singular initial matrix and its application to linear programming
 Zhang   Liwei ;
Abstract
 In this paper the ABS algorithm with singular initial matrix is considered and used to construct an active set algorithm for solving the linear programming prbblem. We prove that this active set algorithm is reduced to the simplex algorithm whep it starts from a vertex or it meets a vertex  
Keywords
ABS Algorithms, Singular Initial Abaffian, Active Set Method, Linear Programming, Simplex Method, Huang Method
