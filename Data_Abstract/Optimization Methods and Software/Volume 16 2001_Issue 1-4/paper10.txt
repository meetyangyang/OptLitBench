ABS solution of a class of linear integer inequalities and integer LP problems
 Hamid   Esmaeili ; Nezam   Mahdavi-Amiri ; Emilio   Spedicato ;
Abstract
 Using the recently developed ABS algorithm for solving linear Diophantine equations we give a representation of the solutions of a system of m linear integer inequalities in n variables, m≤n, with full rank coefficient matrix. We apply this result to solve linear integer programming problems with m≤n inequalities  
Keywords
ABS Algorithm, Diophantine equations, Integer points
