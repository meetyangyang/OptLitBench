Minimal time function with respect to a set of directions: basic properties and applications
Marius Durea;Marian Panţiruc;Radu Strugariu;
Abstract
In this paper we study a special minimal time function, given with respect to a set of directions. Several properties, concerning continuity, convexity, Lipschitz behaviour and subdifferential calculus are explored. An application to a vectorial location problem is provided.
Keywords
minimal time function, convexity, Lipschitz properties, subdifferential calculusAMS Subject Classification: 49J53, 49K27, 90C46
