Large-Scale, Nonlinearly Constrained Optimization on a 1024-Processor nCUBE
J. H.  Glick and J. B.  Rosen;
Abstract
An efficient parallel partition algorithm is described that has been developed for solving a class of large-scale, structured, nonlinear optimization problems having many nonlinear constraints. This class of problems has an objective function and constraints that are linear with respect to a large number of variables and nonlinear with respect to a relatively small number of other variables. Also, the Jacobian of the constraint matrix with respect to the linear variables is block diagonal and dense with respect to the nonlinear variables, so the nonlinear variables are linking variables in an otherwise separable problem. Problems having this structure arise in applications that include combined manufacturing and distribution modeling and stochastic optimization.The partition algorithm first projects the original large-scale problem onto the much smaller space of the nonlinear linking variables. A serial version was implemented on a Cray X-MP and a parallel version on a 1024-processor nCUBE multicomputer. The algorithm was tested on a range of quadratic constraint test problems, the largest having over one million quadratic constraints. The authors’ studies showed that serial implementation gives good solution times when compared to using a general purpose nonlinear programming package or a bundle method. The parallel implementation was able to solve the very large problems with good efficiency.
