Differentiable Piecewise Quadratic Exact Penalty Functions for Quadratic Programs with Simple Bound Constraints
Wu  Li;
Abstract
The quadratic program with simple bound constraints is reformulated as unconstrained minimization of a differentiable piecewise quadratic function. The two problems have the same set of local solutions, the same set of isolated local solutions, and the same set of global solutions. Unlike other penalty functions, a parameter involved in the unconstrained reformulation can be easily determined by the spectrum radius of the Hessian of the objective function in the original quadratic program. Moreover, the exact penalty function can also be derived from the Hestenes–Powell–Rockafellar augmented Lagrangian function for two-sided inequality constrained minimization problems by using Fletcher’s multiplier function.
Keywords
