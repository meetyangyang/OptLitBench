Weighted Sensitivity Minimization in Systems with a Single Input Delay: A State Space Solution
Gilead  Tadmor;
Abstract
A differential/algebraic matrix Riccati equation--based design method is developed. The solution is reached via a parallel, two-way analysis of an underlying differential game: in the framework of both the finite order delay system and a distributed parameter, ordinary evolution model.
Keywords
input delay, Nehari problem, state space solutions, matrix Riccati equations, 49K25, 93B36, 93D21,
