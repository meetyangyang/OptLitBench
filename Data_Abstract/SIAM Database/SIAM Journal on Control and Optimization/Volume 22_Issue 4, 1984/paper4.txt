On the Control Canonical Structure of a Class of Scalar Input Systems
Russell G.  Teglas;
Abstract
In this paper, we demonstrate the equivalence of a scalar input system $\dot x = \mathcal{A}x + \mathfrak{b}u$, for which the eigenvalues of the generator $\mathcal{A}$ coincide with the roots of the entire function \[p(\omega ) = e^{\omega T}  + a_1 e^{\omega (T - \theta _1 )}  +  \cdots  + a_m  + \int_0^T {a(\theta )e^{\omega (T - \theta )} d\theta ,} \] with the controlled scalar functional equation \[y(t) + a_1 y(t - \theta _1 ) +  \cdots  + a_m y(t - T) + \int_0^T {a(\theta )y(t - \theta )d\theta  = u(t)} .\] The theory of nonharmonic Fourier series is then employed to investigate the placement of eigenvalues in the closed-loop system with continuous state feedback.
