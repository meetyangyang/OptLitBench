The Singularly Perturbed Linear State Regulator Problem
R. E.  O’Malley, Jr.;
Abstract
We shall seek the optimal control and corresponding trajectories for the state regulator problem \[\begin{gathered}   \frac{{dx}}{{dt}} = A_1 (t,\varepsilon )x + A_2 (t,\varepsilon )z + B_1 (t,\varepsilon )u, \hfill \\  \varepsilon \frac{{dz}}{{dt}} = A_3 (t,\varepsilon )x + A_4 (t,\varepsilon )z + B_2 (t,\varepsilon )u \hfill \\ \end{gathered}\] on the interval $0 \leqq t \leqq 1$ with the initial state vector $y(0,\varepsilon )$ prescribed for ($y = \left( {\begin{array}{*{20}c}  x  \\   z  \\ \end{array} } \right)} $) and where the quadratic cost functional \[J(\varepsilon ) = \frac{1}{2}y'(1,\varepsilon )\pi (\varepsilon )y(1,\varepsilon ) + \frac{1}{2}\int_0^1 {\left[y'(t,\varepsilon )Q(t,\varepsilon )y(t,\varepsilon ) + u'(t,\varepsilon )R(t,\varepsilon )u(t,\varepsilon )\right]dt} \] is minimized. Under appropriate hypotheses, it will be possible to obtain a complete asymptotic solution uniformly valid as $\varepsilon  \to 0$. Such results are of value in practical control theory where $\varepsilon $ represents certain often neglected “parasitic” parameters whose presence causes the order of the model to increase.
