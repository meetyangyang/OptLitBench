A New Algorithm for a Class of Quadratic Programming Problems with Application to Control
M. D.  Canon and James H.  Eaton;
Abstract
The control problem considered is that of determining an input which will take a linear sampled system from a specified initial state to a desired terminal state in minimum time, subject to amplitude constraints on the input. The problem is reduced to solving a sequence of simple quadratic programming problems; a new algorithm is presented for solving this class of problems. Preliminary computational results for a fourth-order system are favorable.
