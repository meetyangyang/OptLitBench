On an Algorithm for Optimal Control Using Pontryagin’s Maximum Principle
Joseph Frédéric  Bonnans;
Abstract
Y. Sakawa and Y. Shindo recently proposed an algorithm to solve open-loop optimal control problems, using Pontryagin’s maximum principle. It is established here that, under some hypothesis, the algorithm is well-defined and globally converges in some weak sense. When the stepsize of the algorithm tends to zero, the displacements are equivalent to those of a gradient with projection method. If the control enters in a linear way in the state equation and in a quadratic way in the criterion, the algorithm can be interpreted as a gradient plus projection method in some new metric.
