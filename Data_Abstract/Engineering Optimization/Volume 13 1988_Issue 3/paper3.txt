AN APPROACH IN THE DESIGN OF A UNIT-LOAD AUTOMATED CAROUSEL STORAGE SYSTEM
 MOON-KYU   LEE ; HARK   HWANG ;
Abstract
 This paper deals with the design problem of a unit-load automated carousel storage system in which each carousel conveyor is served individually by a single storage/retrieval (S/R) device. We first derive an expected travel time per operation cycle and estimate the number of unit loads processed per unit time of the system. Then, a mathematical model is proposed in a nonlinear integer form from which a set of design variables can be determined optimally. The objective of the model is to minimize initial investment and operating costs over a planning horizon, whilst satisfying a set of constraints imposed upon the system. A solution methodology is developed based on search techniques. The validity of the model is illustrated through an example problem and the sensitivity of the system parameters are also evaluated. 
Keywords
Automated carousel storage system, design, optimization nonlinear integer programming, search procedure
