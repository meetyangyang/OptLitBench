A COMPUTATIONAL ALGORITHM FOR THE SENSITIVITY ANALYSIS OF POSYNOMIAL AND SIGNOMIAL GEOMETRIC PROGRAMMING PROBLEMS
 C.   MOHAN ; VEENA   AGARWAL ;
Abstract
 In this paper we present a computational algorithm which can be used to carry out the sensitivity analysis of posynomial and signomial geometric programming problems, both for changes in the values of the cost coefficients and for changes in the values of the exponents of the unknown variables. Computational experience of using this algorithm to study the sensitivity analysis of some test examples is also presented.
