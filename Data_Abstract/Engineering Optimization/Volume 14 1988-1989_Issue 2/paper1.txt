AUTOMATIC CONSTRAINT GENERATION IN THE DESIGN OF ELECTRICITY DISTRIBUTION NETWORKS
 A. C.   MARSHALL ; T. B.   BOFFEY ;
Abstract
 There are a number of inherent problems in realistically designing a secure minimum cost electricity supply network: these include constraint specification, security assessment, recognition of the optimal design and solution methodology. It is shown here how Lagrangean relaxation techniques can be used to form a Lagrangean dual. Subgradient optimization is then employed to solve the dual. The problem of constraint specification is overcome by the use of automatic security assessment procedures which, as solution progresses, add missing group transfer and derived ‘cutting plane’ constraints to the dual. A stopping criterion is provided which enables the optimal solution to be recognised. The process is automated and a small illustrative example is given. 
Keywords
Electricity network design, electricity network security, Lagrangean relaxation, subgradient optimization
