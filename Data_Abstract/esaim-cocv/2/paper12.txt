Local and global null controllability of time varying linear control systems
F. Colonius1 and R. Johnson;
Abstract
        Fritz.Colonius@Math.uni-augburg.de       AbstractFor linear control systems with coefficients determined by a dynamical system null controllability is discussed. If uniform local null controllability holds, and if the Lyapounov exponents of thehomogeneous equation are all non-positive, then the system is globally null controllable for almost all paths of the dynamical system. Even if some Lyapounov exponents are positive, anirreducibility assumption implies that, for a dense set of paths, the system is globally null controllable. RésuméOn étudie la contrôlabilité à zéro pour des systèmes de contrôle linéaires à coefficients déterminés par un système dynamique. Si on a contrôlabilité à zéro uniforme et si les exposants de Lyapounov de l'équation homogène sont tous négatifs, alors le système est globalement contrôlable à zéro pour presque toutes les trajectoires du système dynamique. Même dans le cas où des exposants de Lyapounov sont positifs, une hypothèse d'irréductibilité implique que, pour un ensemble dense de trajectoires, le système est globalement contrôlable à zéro.
Keywords
Linear control systems, null controllability, time varying systems, stationary ergodic processes.
