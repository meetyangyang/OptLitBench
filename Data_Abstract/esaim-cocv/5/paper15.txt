Value functions for Bolza problems with discontinuous Lagrangians and Hamilton-Jacobi inequalities
Gianni Dal Maso1 and Hélène Frankowska2;
Abstract
We investigate the value function of the Bolza problem of theCalculus of Variationswith a lower semicontinuous Lagrangian L and a final cost ,andshow that it is locally Lipschitz for t>0whenever L is locally bounded. It also satisfiesHamilton-Jacobi inequalities in a generalized sense.When the Lagrangian is continuous, then the value function is theunique lower semicontinuous solutionto the corresponding Hamilton-Jacobi equation, while for discontinuousLagrangian we characterize the value function by using the socalled contingent inequalities.
Keywords
Discontinuous Lagrangians, 
 Hamilton-Jacobi equations, viability theory, viscosity solutions
