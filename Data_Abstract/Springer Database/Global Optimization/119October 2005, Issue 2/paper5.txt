Generalized Nonlinear Lagrangian Formulation for Bounded Integer Programming
Yifan Xu;Chunli Liu;Duan Li;
Abstract
Several nonlinear Lagrangian formulations have been recently proposed for
bounded integer programming problems. While possessing an asymptotic strong duality
property, these formulations offer a success guarantee for the identification of an optimal
primal solution via a dual search. Investigating common features of nonlinear Lagrangian
formulations in constructing a nonlinear support for nonconvex piecewise constant perturbation function, this paper proposes a generalized nonlinear Lagrangian formulation of which many existing nonlinear Lagrangian formulations become special cases.

Keywords
duality gap  integer programming  Lagrangian relaxation  nonlinear integer programming  nonlinear Lagrangian formulation
