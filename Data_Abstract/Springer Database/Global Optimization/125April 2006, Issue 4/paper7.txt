The Essential Component of the Solution Set for Vector Equilibrium Problems
Qun Luo;
Abstract
In the paper, by using Ky Fan’s section theorem, we obtain an existence theorem for vector equilibrium problems. Motivated by the ideas of Kinoshita and McLennan, we introduce the concept of the essential component of the solution set for vector equilibrium problems, and we prove that there exists at least one essential component of the solution set for every vector equilibrium problem satisfying some conditions.
Keywords
C- function  essential component  vector equilibrium problem
