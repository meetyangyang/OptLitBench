Optimal decision for the target value of performance-based structural system reliability
G. Li;G. Cheng;
Abstract
The optimal decision model of the target value of performance-based structural system reliability is established according to the cost-effectiveness criterion in the present paper. Three types of optimal decision models are studied: the conceptive model, the parameter programming model and the constraint relaxation model. The conceptive model is inoperable because the objective function must be expressed as the function of the structural target reliability. The parameter programming and constraint relax models are proved to be equivalent with Kuhn–Tucker conditions when the performance-based structural system reliability constraints are all active. Finally the numerical examples of RC-frames for different earthquake-resistant performance are calculated.
Keywords
Key words: structural system reliability, performance-based design, optimal decision, target reliability
