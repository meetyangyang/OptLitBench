Effective structural optimization based on equivalent static loads combined with system reduction method
Euiyoung Kim;Hyun-gi Kim;Sungmin Baek;Maenghyo Cho;
Abstract
The present study proposes an efficient equivalent static load (ESL) method for dynamic optimization problems. The ESL method transforms dynamic problems into equivalent static problems. Although it provides an efficient way to solve dynamic optimization problems, large-scale dynamic problems still require heavy computation. In order to approximate solutions to dynamic optimization problems more efficiently, the proposed method combines ESLs with a system reduction method. The proposed method combines transient dynamic analysis with static optimization to form a highly efficient dynamic optimization algorithm.
Keywords
System reduction method  Equivalent static loads  Structural optimization  Dynamic response optimization
