Shape optimization for suppressing brake squeal
Kohei Shintani;Hideyuki Azegami;
Abstract
The present paper describes a solution to a non-parametric shape optimization problem of a brake model to suppress squeal noise. The brake model consists of a rotor and a pad, between which Coulomb friction occurs. The main problem is defined as a complex eigenvalue problem of the brake model obtained from the equation of motion. As an objective cost function, we use the positive real part of the complex eigenvalue generating the brake squeal. The volume of the pad is used as a constraint cost function. The Fréchet derivative of the objective cost function with respect to the domain variation, which we refer to as the shape derivative of the objective cost function, is evaluated using the solution of the main problem and the adjoint problem. A scheme by which to solve the shape optimization problem using an iterative algorithm based on the H 1 gradient method (the traction method) for reshaping is presented. Numerical results obtained using a simple rotor-pad model reveal that the real part of the target complex eigenvalue decreases monotonically, thus satisfying the volume constraint.
Keywords
