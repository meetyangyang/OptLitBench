Reconstructing a matrix from a partial sampling of Pareto eigenvalues
Pedro Gajardo;Alberto Seeger;
Abstract
Let Λ={λ 1,…,λ  p } be a given set of distinct real numbers. This work deals with the problem of constructing a real matrix A of order n such that each element of Λ is a Pareto eigenvalue of A, that is to say, for all k∈{1,…,p} the complementarity system 
$$x\geq \mathbf{0}_n,\quad Ax-\lambda_k x\geq \mathbf{0}_n,\quad \langle x, Ax-\lambda_k x\rangle = 0$$ admits a nonzero solution x∈ℝ n .
Keywords
Cone-constrained eigenvalue problem  Pareto spectrum  Inverse Pareto eigenvalue problem  Newton method  Normal flow algorithm  Underdetermined system of equations
