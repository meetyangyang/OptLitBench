An Efficient Algorithm for the Knapsack Sharing Problem
Mhand Hifi;Slim Sadfi;Abdelkader Sbihi;
Abstract
The Knapsack Sharing Problem (KSP) is an NP-Hard combinatorial optimization problem, admitted in numerous real world applications. In the KSP, we have a knapsack of capacity c and a set of n objects, namely N, where each object j, j = 1,...,n, is associated with a profit pj and a weight wj. The set of objects N is composed of m different classes of objects Ji, i = 1,...,m, and N = ∪ m i=1Ji. The aim is to determine a subset of objects to be included in the knapsack that realizes a max-min value over all classes.
Keywords
