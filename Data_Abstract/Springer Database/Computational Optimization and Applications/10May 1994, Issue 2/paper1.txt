Reducing the number of variables in Integer and Linear Programming Problems
Djangir A. Babayev;Sahib S. Mardanov;
Abstract
Computational difficulties in solving the Integer Programming Problems (IPP) are caused to a considerable degree by the number of variables. If the number of variables is small, then even NP-complete problems usually can be solved with a reasonable expenditure of effort.
Keywords
Linear  Integer  Mixed Integer Programming Problems  integer variables  number of variables  a priori analysis
