Classification of companies using maximal margin ellipsoidal surfaces
Hiroshi Konno;Masato Saito;
Abstract
We recently proposed a data mining approach for classifying companies into several groups using ellipsoidal surfaces. This problem can be formulated as a semi-definite programming problem, which can be solved within a practical amount of computation time by using a state-of-the-art semi-definite programming software. It turned out that this method performs better for this application than earlier methods based on linear and general quadratic surfaces.
Keywords
