A note on absolute value equations
Sheng-Long Hu;Zheng-Hai Huang;
Abstract
In this note, we reformulate a system of absolute value equations (AVEs) as a standard linear complementarity problem (LCP) without any assumption. Utilizing some known results for the LCP, existence and convexity results for the solution set of the AVE are proposed.
Keywords
Absolute value equations  Linear complementarity problem
