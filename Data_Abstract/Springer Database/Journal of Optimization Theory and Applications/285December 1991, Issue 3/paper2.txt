Constrained control for uncertain linear systems
F. Blanchini;
Abstract
The linear state feedback synthesis problem for uncertain linear systems with state and control constraints is considered. We assume that the uncertainties are present in both the state and input matrices and they are bounded. The main goal is to find a linear control law assuring that both state and input constraints are fulfilled at each time. The problem is solved by confining the state within a compact and convex positively invariant set contained in the allowable state region.
Keywords
Stabilization  uncertain systems  constrained control  positive invariance  linear programming
