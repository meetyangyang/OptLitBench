A double-stage piecewise recursive filter for real-time applications
J. N. Damoulakis;G. Cocharo;
Abstract
This paper examines the basic problem of estimating the state of a system described by a set of dynamical equations. The state estimation is performed by means of optimal filtering techniques in which the state observation is given by a set of nonlinear equations.
