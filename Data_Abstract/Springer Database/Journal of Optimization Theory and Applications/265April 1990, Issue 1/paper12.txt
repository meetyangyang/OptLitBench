Solution of parametrized linear inequalities by Fourier elimination and its applications
S. S. Keerthi;K. Sridharan;
Abstract
The parametric solution of a linear system of inequalitiesAx≤Bb, with parameterb, is considered. Fourier elimination is used to give a facial representation for the set ofb-values for which the system is consistent. Some interesting applications of the problem are discussed. Although the worst case complexity of the method is an exponential function of the size ofA, the computations are intuitive and very simple.
Keywords
Linear inequalities  polyhedra  linear programming  Fourier elimination
