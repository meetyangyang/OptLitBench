Saddle-point type optimality criteria for generalized fractional programming
Z. K. Xu;
Abstract
Generally speaking, one cannot expect that there exists a Lagrangian saddle point even for a linear fractional program. Using the functionsGF(x,r0,r) andGK(x,u) somewhat like the Lagrangian functions, we present saddle-point type optimality criteria for generalized fractional programming under carefully selected assumptions. In addition, we point out conditions that ensure that a local optimal solution of the program mentioned is global.
Keywords
Generalized fractional programming  saddle-point type optimality criteria  local optimal solutions  global optimal solutions
