Connectedness of the efficient set in strictly quasiconcave vector maximization
Y. D. Hu;E. J. Sun;
Abstract
In this paper, the concepts of quasiconcave set and strictly quasiconcave set are introduced. By using these concepts, we get a new sufficient condition for the efficient outcome set to be connected. This leads to the connectedness of the efficient solution set in strictly quasiconcave vector maximization under the mild condition that the efficient frontier is closed.
Keywords
Vector maximization  strictly quasiconcave sets  connectedness
