An outer approximations algorithm for computer-aided design problems
D. Q. Mayne;E. Polak;R. Trahan;
Abstract
This paper presents an implementable algorithm of the outer approximations type for solving nonlinear programming problems with functional inequality constraints. The algorithm was motivated by engineering design problems in circuit tolerancing, multivariable control, and shock-resistant structures.
Keywords
Nonlinear programming  mathematical programming  outer approximation algorithm  computer-aided design  design problems
