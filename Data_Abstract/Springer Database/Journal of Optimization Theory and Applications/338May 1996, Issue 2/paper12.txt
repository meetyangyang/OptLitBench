On the connectedness of the efficient set for strictly quasiconvex vector minimization problems
E. J. Sun;
Abstract
In this paper, we investigate the connectedness of the efficient solution set for vector minimization problems defined by a continuous vector-valued strictly quasiconvex functionf=(f1,...,f m ) T  and a convex compact setX. It is shown that the efficient solution set is connected if one component off is strongly quasiconvex onX.
Keywords
Vector minimization  efficient solution set  connectivity  generalized convexity
