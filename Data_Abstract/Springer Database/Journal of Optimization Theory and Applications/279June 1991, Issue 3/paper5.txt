Search game in a rectangle
A. Yu. Garnaev;
Abstract
We consider a search game in which the searcher (player S) moves along a continuous trajectory in a rectangleQ. The velocity vectogram of player S is a rhombus-type set. In this paper, we construct the strategies of both players which make it possible to find the asymptotic value of the game in the case of small discovery radius.
Keywords
Search games  princess and monster games  optimal strategies  value of the game
