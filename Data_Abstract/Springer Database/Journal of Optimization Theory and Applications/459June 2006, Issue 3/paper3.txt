Decentralized Guaranteed Cost Control for Uncertain Large-Scale Systems Using Delayed Feedback: LMI Optimization Approach
O. M. Kown;J. H. Park;
Abstract
In this paper, we propose a design method of guaranteed cost controllers for uncertain large-scale systems with time delays in subsystem interconnections using delayed feedback. Using the Lyapunov method, a linear matrix inequality (LMI) optimization problem is formulated to design a delayed feedback controller which minimizes the upper bound of a given quadratic cost function. A numerical example is included to illustrate the design procedures.
Keywords
Uncertain large-scale systems  delayed feedback  guaranteed cost control  linear matrix inequalities  Lyapunov method
