On existence of optimal controls
A. M. Steinberg;H. L. Stalford;
Abstract
An optimal control is shown to exist for a system\(\dot x = f(x,t,u)\) when the Hamiltonian is a strictly convex function of the control. It is proven that a system satisfying this condition must have state equations that are linear in the control and a cost functional whose integrand is strictly convex in the control.
Keywords
Convex Function  State Equation
