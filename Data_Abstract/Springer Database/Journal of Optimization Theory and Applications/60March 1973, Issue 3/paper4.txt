Necessary optimality criteria in mathematical programming in normed linear spaces
M. S. Bazaraa;J. J. Goode;
Abstract
In this study, we develop optimality criteria for a mathematical programming problem. The constraints are defined by an arbitrary set as well as infinitely many equality and inequality constraints. Necessary conditions of the Kuhn-Tucker type are obtained.
Keywords
Programming Problem  Linear Space  Mathematical Programming  Optimality Criterion  Inequality Constraint
