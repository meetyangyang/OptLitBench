Density of the set of positive proper minimal points in the set of minimal points
X. H. Gong;
Abstract
One of the important problems of vector optimization concerns the density of the set of positive proper minimal points in the set of minimal points. We use the concepts of dentable point and approximating cones to derive sufficient conditions guaranteeing that the set of minimal points is contained in the closure of the set of positive proper minimal points. The result can be applied to obtain a density result for the unit ball in 1 p , 1<p<+∞, which does not follow from any other well-known density theorem.
Keywords
