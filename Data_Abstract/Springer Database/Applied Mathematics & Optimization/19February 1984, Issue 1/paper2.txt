A noncooperativen-person semi-Markov game with a separable metric state space
Hang-Chin Lai;Kensuke Tanaka;
Abstract
A noncooperativen-person semi-Markov game with a separable metric state space is studied in this paper. We define the equilibrium condition of the noncooperative game, and find a necessary and sufficient condition for which the equilibrium condition holds, and prove that in our game system there possesses an equilibrium point so that each player has his own stationary equilibrium strategy.
Keywords
Equilibrium Condition  State Space  System Theory  Equilibrium Point  Mathematical Method
