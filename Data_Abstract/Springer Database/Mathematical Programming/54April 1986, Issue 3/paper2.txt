Linear complementarity problems with an invariant number of solutions
Richard E. Stone;
Abstract
This paper studies the class INS of all realn × n matricesM for which the linear complementarity problem (q, M) has exactlyk solutions—k depending only onM—for all realn-vectorsq interior to the coneK(M) of vectors for which (q, M) has any solution at all. This generalizes the results in Cottle and Stone (1983) which deal with the subclassU in INS wherek equals one.
