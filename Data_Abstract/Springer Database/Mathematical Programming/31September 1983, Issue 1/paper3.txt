The best parameter subset using the Chebychev curve fitting criterion
Ronald D. Armstrong;Philip O. Beck;
Abstract
The Chebychev (also Minimax andL ∞ Norm) criterion has been widely studied as a method for curve fitting. Published computer codes are available to obtain the optimal parameter estimates to fit a linear function to a set of given points under the Chebychev criterion. The purpose of this paper is to study procedures for obtaining the best subset ofq parameters from a given set ofm parameters whereq is less-than-or-equal-tom.
