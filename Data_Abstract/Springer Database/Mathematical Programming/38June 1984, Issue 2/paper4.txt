A class of rank-two ellipsoid algorithms for convex programming
A. Ech-Cherif;J. G. Ecker;
Abstract
In this paper, we develop a method for constructing minimum volume ellipsoids containing a wedge-shaped subset of a given ellipsoid. This construction yields a class of ellipsoid algorithms for convex programming that use rank-two update formulae.
Keywords
Ellipsoid Algorithms  Rank-Two Updates  Convex Programming
