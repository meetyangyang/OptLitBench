A modification of Armijo's step-size rule for negative curvature
Garth P. McCormick;
Abstract
Armijo's step-size procedure for function minimization is modified to include second derivative information. Accumulation points using this procedure are shown to be stationary points with positive semi-definite Hessian matrices.
Keywords
Unconstrained minimization  Step size procedures  Negative curvature  Second order convergence  Line search  Newton's method
