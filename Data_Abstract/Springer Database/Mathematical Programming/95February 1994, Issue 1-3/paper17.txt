A two-stage successive overrelaxation algorithm for solving the symmetric linear complementarity problem
Karen T. Medhi;
Abstract
We propose a two-stage successive overrelaxation method (TSOR) algorithm for solving the symmetric linear complementarity problem. After the first SOR preprocessing stage this algorithm concentrates on updating a certain prescribed subset of variables which is determined by exploiting the complementarity property. We demonstrate that this algorithm successfully solves problems with up to ten thousand variables.
Keywords
SOR  Symmetric linear complementarity problem  Large-scale programming
