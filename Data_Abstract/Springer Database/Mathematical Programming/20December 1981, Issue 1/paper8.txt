Degenerate complementary cones induced by aK0-matrix
S. R. Mohan;
Abstract
In this paper we prove two results concerning the unionC of all the degenerate complementary cones associated with the linear complementarity problem (M, q) whereM is aK0-matrix.(1) C is the same as the set of allq ∈Rn for which (M, q) has infinitely many solutions.  (2) C is the same as the boundary of the set of allq ∈ Rn for which (M, q) has a solution, an easily observable geometric result for a 2 × 2K0-matrix.
