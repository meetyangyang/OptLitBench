A randomized algorithm for fixed-dimensional linear programming
M. E. Dyer;A. M. Frieze;
Abstract
We give a (Las Vegas) randomized algorithm for linear programming in a fixed dimensiond for which the expected computation time is\(O(d^{(3 + \varepsilon _d )d} n)\), where limd→∞ε d  = 0. This improves the corresponding worst-case complexity,\(O(3^{d^2 } n)\). The method is based on a recent idea of Clarkson. Two variations on the algorithm are examined briefly.
