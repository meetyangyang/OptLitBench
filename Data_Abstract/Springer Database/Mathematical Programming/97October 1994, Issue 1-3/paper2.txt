An infeasible-interior-point algorithm for linear complementarity problems
Stephen J. Wright;
Abstract
We modify the algorithm of Zhang to obtain anO(n2L) infeasible-interior-point algorithm for monotone linear complementarity problems that has an asymptoticQ-subquadratic convergence rate. The algorithm requires the solution of at most two linear systems with the same coefficient matrix at each iteration.
Keywords
