Global error bounds with fractional exponents
Kung Fu Ng;Xi Yin Zheng;
Abstract
Using the partial order induced by a proper weakly lower semicontinuous function on a reflexive Banach space X we give a sufficient condition for f to have error bounds with fractional exponents. Application is given to identify the set of such exponents for quadratic functions.
Keywords
Key words: error bound – lower semicontinuous function – convex function – quadratic function – reflexive Banach space Mathematics Subject Classification (1991): 90C31, 90C25, 49J52
