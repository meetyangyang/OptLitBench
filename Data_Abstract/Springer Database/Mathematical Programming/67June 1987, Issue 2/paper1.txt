Some NP-complete problems in quadratic and nonlinear programming
Katta G. Murty;Santosh N. Kabadi;
Abstract
In continuous variable, smooth, nonconvex nonlinear programming, we analyze the complexity of checking whether(a) a given feasible solution is not a local minimum, and  (b) the objective function is not bounded below on the set of feasible solutions.  
Keywords
Nonconvex nonlinear programming  local minimum  global minimum  copositive matrices  NP-complete
