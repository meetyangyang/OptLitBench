Toeplitz type theorems for double sequences and their applications to some iterative process in zero-sum continuous games
   A.   Cegielski ;
Abstract
 In the paper there is considered some version of the Toeplitz theorem. In particular there is proved some appropriate version of this theorem for two-dimensional sequences. There are also given connections between these theorems and some iterative process in two person zero-sum games with continuous pay-off function 
Keywords
Toeplitz Theorem, Zero-Sum Srame
