Zum spieltheoretlschen kompromiß in der vektoroptimierung
   G.   Dewess ;
Abstract
 Generally, the use of the game -theoretical approach according to Jüttler[8] for linear programming with more than one objective functions leads to a nonefficient solution. We precise a natural, more sophisticated version of that approach (essentially, the same as described by Köbth [10]) so that efficiency follows. Using a parametric version (game-theoretical compromise with differently weighted coefficients) which is goal-programming with a special maximum norm we can compare this with other types of compromises. The use of parameters in the described way is more evident and better qualified for a dialog between man and computer. With assumptions not being decisive for applications a new theorem on efficiency holds.  The statements on existence and efficiency can be generalized to other vector programming problems (compact region, objective functions continuously, on assumptions on convexity).
