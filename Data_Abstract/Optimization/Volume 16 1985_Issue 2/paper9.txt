Lökal optimale lösungen der kombinierten strom- und spannungsoptimlerung auf netzwerken im anwendungsfall der abaufplanung
   G.   Nägler ;
Abstract
 A mathematical problem is defined, which is a combination between the classical network flow problem with the classical potential problem. As a branch of application is presented a model for the quantitative and temporal control of resource flows. Because the problem is NP-hard, the following text deals with the obtainment of only local optimal solutions. It is demonstrated, that it is possible to transform the known algorithm of network optimization. The sketched algorithm of solution is demonstrated by an example.
