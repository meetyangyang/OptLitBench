Probabilistic and deterministic local search for solving the binary multiknapsack problem
   M.   Bertocchi ;   A.   Butti ;   L.   Słomiñ ski ;   J.   Sobczyñska ;
Abstract
 Results of extensive computational experiments aimed at comparing performance quality (accuracy of an approximate solution and the running time) of the Threshold Accept (TA) approach versus the Simulated Annealing (SA) and a Greedy-Local improvement Search (GLS) approaches are presented. This comparison is done on a large set of test examples that represent the binary Multiknapsack Problem (MKP). It appears that the best accuracy results are obtained for the TA algorithm. Purely deterministic search is only slightly worse in accuracy, but remarkably better in the running time, comparing to both SA and TA  
Keywords
Multiknapsack Problem, Simulated Annealing and Threshold Accept Algorithms, Algorithms, Deterministic Greedy-Local Type Search
