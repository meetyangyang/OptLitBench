Characterizations of set relations with respect to variable domination structures via oriented distance function
Qamrul Hasan Ansari;Elisabeth Köbis;Pradeep Kumar Sharma;
Abstract
In this paper, we characterize several known set relations with respect to variable domination structures by defining scalarization functions in terms of the oriented distance function. The characterization of minimal elements of a family of sets and the properties of sets of minimal elements are discussed by means of scalarization functionals. An application of variable domination structures to uncertain vector optimization is also given.
Keywords
Oriented distance function, set optimization, set relations, variable domination structures, optimality conditionsAMS Subject Classifications: 49J53, 90C29
