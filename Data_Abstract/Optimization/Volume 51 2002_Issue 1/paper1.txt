Backward Error Analysis for the Linear Programming Problem
 Marie-Cécile   Darracq-Calmettes ; Jean-Pierre   Dedieu ;
Abstract
 An approximation of the solution of a linear programming problem is described as the exact solution of a nearby problem. To compute this one, we use a Sylvester equation. Next we give the distance between these two problems. 
Keywords
Linear Programming, Sylvester Equation, Backward Error
