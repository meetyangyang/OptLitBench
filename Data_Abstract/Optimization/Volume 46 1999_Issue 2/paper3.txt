Pontryagin’s Maximum Principle for DieudonnÉ-Rashevsky Type Problems Involving Lipschitz Functions
 M.   Wagner ;
Abstract
 We prove an ∊-maximum principle for Dieudonné-Rashevsky type problems involving Lipschitz state variables by means of a duality approach. The conditions will be parameter-free if the set of admissible controls possesses interior points. Finally, using the Yosida-Hewitt decomposition of the multipliers, we arrive at a point wise formulated maximum condition  
Keywords
Dieudonné-Rashevsky Type Problems, Generalized Pontryagin’s Maximum Principle, Lipschitz Functions, Finitely Additive Set Functions
