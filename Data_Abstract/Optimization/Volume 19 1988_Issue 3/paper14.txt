  

Abstract
 The turnpike set is of great importance for the theory of economical dynamics. This set is a minimal closed subset of the space to which all paths converge having a maximal growth factor. The most interesting situation in applications is this one if a turnpike set is a ray. Then it is possible to form a path arbitrary close to the optimal one from any initial state. As it is shown in this paper this situation takes place for nearly all continuous models defined in a certain function space.
