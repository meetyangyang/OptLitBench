On an inverse problem of entropy othimization and related questions
   B.   Fritzsche ;   S.   Fuchs ;   B.   Kirstein ;
Abstract
 In a fundamental paper Arov and Krein studied entropy minimization problems over a set of matrix-valued Schur functions which arises from a fractional linear transformation the generating matrix-valued function of which is a jpq -inner function.Our paper handles the corresponding inverse problem: If a p x q Schur function f is given, then the problem is to determine all jpq -inner functions for which f turns out to be the associatedentropy-minimal function  
Keywords
Stationary Sequences, Prediction Theory, Entropy, Schur Functions,  J-inner Functions
