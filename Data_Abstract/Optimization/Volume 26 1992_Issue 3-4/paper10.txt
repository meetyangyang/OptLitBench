Contiuous time inverse vector optimization problems
   R.N.   Kaul ;   D.   Bhatia ;   S.   Aggarwal ;
Abstract
 In this paper we present a continuous time generalization of nonlinear inverse vector optimization problem. It is shown that finding a solution to inverse vector optimization problem which is efficient with respect to the constraints as well as to the objectives is equivalent to solving a single scalar valued continuous time programming problem. Finally, as a special case, continuous linear inverse vector optimizaton problem is considered.  
Keywords
Continuous inverse vector optimization, linear, nonlinear problems
