The complexity of a special convex programming problem connected with nonlinear optimization
   E.I.   Nenakhovt ;   M.E.   Primak ;
Abstract
 In this paper, an algorithm for solving a special convex programming problem (CPP) is proposed. We consider a CPP with an objective function whose values and gradients are not easy to calculate. Algorithm for infinite feasible sets is developed.  
Keywords
Convex Programming Problem, Circumscribed and Inscribed Ellipsoid Methods, Computational Cost
