On characterizing solution set of non-differentiable η-pseudolinear extremum problem
 Ke   Quan   Zhao ; Li   Ping   Tang ;
Abstract
 In this article, some basic characterizations of a non-differentiable η-pseudolinear function are obtained. By means of these basic characterizations, the solution set of the non-differentiable η-pseudolinear extremum problem is characterized. Our study improves naturally and extends some previously known results. 
Keywords
