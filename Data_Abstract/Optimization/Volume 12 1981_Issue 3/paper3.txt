On variable dimension algorithms and algorithms using primitive sets
 Hoang   Tuy ;
Abstract
 We discuss the basic scheme and convergence conditions of variable dimension algorithms for computing fixed points and show the analogy of the recent algorithm of van der Laan and Talman with a restart) algorithm using primitive sets that was devel-oped earlier by Tuy-Thoai-Muu. Furthermore, we show that to each algorithm of the class developed by van der Laan and Talman one can associate an algorithm using primi-tive sets wnich proceeds according to the same scheme but with a different triangulation.
