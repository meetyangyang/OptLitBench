On the number of feasible schedules of the open-shop-problem-an application of special latin rectangles
   H.   BrÄSel ; M.   Kleinau ;
Abstract
 The modelling of shop-problems with latin rectangles allows to make some interesting statements about the number of feasible schedules of the so called open-shop-problem [n, m, 0, C max].  At first we investigate the applicability of well-known combinatorial results about latin rectangles to the above problem. Furthermore we give a new enumeration algorithm for schedules. By implementing this method we exactly determine the number of feasible schedules for small n and m. This enumeration method also allows to determine lower bounds for the number of schedules.  
Keywords
Schedulings, Latin Rectangles, Number Problem
