Computing optimal incentives via bilevel programming
   S.   Dempe ;
Abstract
 A model of principal-agency theory is considered which can be formulated as a bilevel programming problem. Here, the agent selects his decision by solving the so-called lower-level problem, which is a convex parametric optimization problem. For control of the agent's actions the principal uses a system of incentives defining the parameter vector for the lower-level problem which should be calculated such that the principal's goal is satisfied as best as possible. An iterative procedure solving the problem of the principal is given and analyzed. This algorithm uses the directional derivative of the optimal solution of the parametric lower-level problem for computing a direction of ascent.
