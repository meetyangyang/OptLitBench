Ein algorithmic zur lösung einer klasse linearer 0–1-optimierungsaufgaben mit spezieller struktur
   W.   Lyska ;   H.   Müller ;
Abstract
 In this work an algorithm is presented to solve the zero-one programming problem with special additional constraints. Rules for the unique generation of index sets  corresponding to the vectors x ∊ R n, x j∊{0,1}, j = 1(1) n are given. Thereby only such index sets are generated, which fulfill the additional constraints. The number of generated index sets is reduced by means of tests, which suit to the problem.
