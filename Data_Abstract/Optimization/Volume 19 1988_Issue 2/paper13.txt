Minimization technique for a convex function with application to multiple regression model
 Wansoo T.   Rhee ;   K.   Anthony Rhee ;
Abstract
 This paper develops an algorithm for estimating the parameters in a general multiple regression model, The estimator coincides with the maximum likelihood estimator when the errors have a probability density function of the type f(t) = C 1 exp ( −φ(t)), where φ is a convex and symmetric function but not necessarily differentiable. Even in the special case corresponding to l 1-estimation, this algorithm is as efficient as the algorithms developed to this date.
