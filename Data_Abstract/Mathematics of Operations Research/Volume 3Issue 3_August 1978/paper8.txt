Attainable Sets of Quasiconcave Markets, II: Convexifiable Sets
Robert James Weber;
Abstract
A set is convexifiabte if there exists a strictly increasing, continuous rescaling of the coordinate axes which makes the set convex. Several classes of sets are investigated with regard to this property. It is shown that every convexifiable compactly generated set is the attainable set of a market in which the traders have quasiconcave utility functions.
