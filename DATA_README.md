# OptLitBench Data Notice

This repository includes bibliographic **abstract text** collected from publicly available optimization literature metadata. The abstract corpus is provided to support **academic research, benchmarking, and reproducibility** for the OptLitBench project.

## What is included

- `Data_Abstract/` — plain-text abstract records organized by journal and publication period
- Derived tables in `results/` — topic weights, metrics, and figures computed from the corpus

## Usage restrictions

1. **Research and education only.** Do not use the corpus for commercial redistribution or republishing as a standalone dataset product without independent legal review.
2. **Publisher rights remain with the original rights holders.** Abstracts may be subject to publisher or platform terms of use.
3. **No warranty.** The corpus is provided as-is for reproducible experimentation. Metadata (year/month) is inferred from directory naming and may contain errors.
4. **Attribution.** If you use OptLitBench, cite the project paper/repository and respect the original journal sources.
